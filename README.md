## Jira-to-Jira Synchronisation

### Prerequisites

- Java 8

- Access to [Jira REST API][jira-rest-api]

### Building

On Unix:

```
./gradlew build
```

On Windows:


```
gradlew.bat build
```


[jira-rest-api]: https://docs.atlassian.com/jira/REST/cloud/
